This is a game that I made in an afternoon following Godot's guide on getting started with ones own 2D game. I took the time to follow this tutorial because I wanted exposure to the different concepts surroundimg game development. 

This is a simple and fun way to have gotten exposure to the ecosystem, the game is a simple survival game where you avoid the "enemies" being the player.

Video games have always been a passion of mine, I am challening myself to build a 2D-Platform fighting game, most similar to Super Smash Bros and Brawlhalla. 

The release contains two available binaries for download, Windows and Linux. Please enjoy!! There is so much more to come!
